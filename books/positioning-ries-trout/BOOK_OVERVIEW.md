# BOOK_OVERVIEW — 《定位》

## 0. 元信息

- 书名：《定位》（Positioning）
- 作者：阿尔·里斯、杰克·特劳特
- 文本来源：用户上传 PDF，116 页；PDF 封面显示“中国财经经济出版社”；前言说明该版本基于 20 周年纪念版翻译。
- 蒸馏目标：抽取可用于品牌、产品、公司、服务、个人职业方向的定位判断方法。

## 1. 一句话主旨

在传播过度、产品过剩的市场里，成功不是把产品讲得更多，而是在预期客户头脑中占据一个清晰、狭窄、可长期坚持的位置。

## 2. 整书结构

1. **心智前提**：社会传播过度，大脑会过滤信息，只接受与既有经验相容的东西。
2. **进入心智**：最有效的方式是成为第一；如果不能成为第一，就借助已有阶梯或重新划分类别。
3. **竞争位置**：领导者、防御者、跟随者、挑战者应采用不同定位策略。
4. **表达载体**：名称、口号、品类词、价格、渠道都不是装饰，而是定位容器。
5. **常见陷阱**：产品延伸、无名缩写、由内而外、人人满意、只说“更好”。
6. **项目方法**：用六个问题推进定位项目，并靠长期一致性累积心智资产。

## 3. 核心概念词典

| 概念 | 蒸馏解释 | 应用提示 |
|---|---|---|
| 定位 | 在预期客户头脑中占据位置 | 先问客户怎么想，再问产品怎么说 |
| 传播过度 | 信息太多导致大脑主动过滤 | 传播要少、准、强 |
| 心智阶梯 | 客户按品类记住品牌的默认排序 | 先画阶梯再定策略 |
| 第一 | 第一个进入心智的位置 | 做不了大品类第一，就做小品类第一 |
| 对比定位 | 借领导者或已有概念建立自己的位置 | “第二但更努力”“非可乐” |
| 重新定位 | 先动摇对手旧认知，再让自己进入 | 不是骂人，是制造认知空位 |
| 名字 | 挂上心智阶梯的钩子 | 名字要能触发定位 |
| 产品延伸 | 用老品牌名覆盖新产品 | 短期省钱，长期可能稀释心智 |
| 由外而内 | 从客户心智出发 | 避免公司自嗨 |
| 坚持 | 定位靠时间累积 | 战略少变，战术常新 |

## 4. 批判性理解

《定位》的强项是把营销从“产品优点表达”转向“客户心智占位”，尤其适合噪音大、竞品多、差异难讲的市场。它的盲点也很明显：书中很多案例来自大众消费品和传统广告时代，对今天的 B2B、医疗器械、招投标、私域内容、搜索和算法推荐场景，需要补充客户旅程、证据链、渠道触点和合规要求。

因此，本蒸馏包把《定位》理解成“心智战略底层工具”，而不是完整的市场进入方案。实际应用时，要把定位与产品事实、临床证据、销售渠道、价格策略和合规边界配合使用。

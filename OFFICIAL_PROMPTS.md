# OFFICIAL_PROMPTS｜《定位》AI Skill 官方提示词

这些提示词适合直接复制到 ChatGPT、Claude、Gemini、Codex 或 Claude Code 中使用。

使用前建议先让 AI 读取：

```text
books/positioning-ries-trout/INDEX.md
```

---

## 0. 通用调用提示词

```text
请先读取 books/positioning-ries-trout/INDEX.md，判断我的任务应该调用哪些《定位》skill。

请不要直接输出普通文案。先完成定位判断，再给出具体表达。

输出时请包含：
1. 应调用的 skill；
2. 客户脑中已有认知；
3. 当前竞争位置；
4. 可占据的心智空位；
5. 应该舍弃的信息；
6. 应该反复强化的信息；
7. 最终建议；
8. 不适用边界。
```

## 1. 市场 / 公众号文章

### 1.1 公众号选题定位

```text
请用《定位》skill 帮我优化这个公众号选题。

目标读者：{例如 ICU 护士长 / 急诊医生 / 经销商 / 产品经理}
原始选题：{填写选题}
产品或主题：{填写产品/主题}
希望达到的目的：{科普 / 转化 / 招商 / 品牌曝光 / 客户教育}

请输出：
1. 目标读者脑中已有的认知；
2. 这个选题是否有清晰心智位置；
3. 它现在最像哪类常见文章；
4. 如何换一个更容易被记住的角度；
5. 5 个标题；
6. 最推荐标题和理由；
7. 文章开头 150 字以内示例。
```

### 1.2 文章标题压缩

```text
请调用 simple-message-compression，帮我压缩这组公众号标题。

目标读者：{填写读者}
文章核心观点：{填写核心观点}
原标题：
{粘贴标题}

请输出：
1. 原标题的问题；
2. 哪些词太空泛；
3. 哪个词最有心智钩子；
4. 10 个更清晰的标题；
5. 最推荐的 1 个。
```

## 3. 产品专员 / 产品经理

### 3.1 新产品定位判断

```text
请调用 six-step-positioning-project，帮我做一次新产品定位判断。

产品：{产品名称}
目标客户：{客户群体}
主要竞品：{竞品列表}
我们的优势：{优势列表}
我们的劣势：{劣势列表}
渠道情况：{直销 / 经销 / 招投标 / 海外代理}

请输出：
1. 我们所在的品类阶梯；
2. 客户脑中可能的前三个品牌；
3. 我们是否有机会成为某个细分第一；
4. 3 个可选定位方向；
5. 每个方向的风险；
6. 最推荐方向；
7. 一句话定位表达。
```

### 3.2 功能卖点取舍

```text
请用 simple-message-compression 和 outside-in-position-diagnosis，帮我判断产品资料里的卖点应该怎么取舍。

目标客户：{客户}
产品：{产品}
卖点列表：
{粘贴卖点}

请输出：
1. 必须保留的 1 个核心卖点；
2. 可以作为辅助证明的 3 个卖点；
3. 建议删除或弱化的卖点；
4. 为什么这些卖点不适合放在主传播位；
5. 最终产品资料第一页应该怎么写。
```

### 3.3 产品延伸判断

```text

新产品：{新产品}
新产品目标客户：{客户}
新产品和老产品的关系：{相同品类 / 相邻品类 / 完全新品类}

请输出：
1. 沿用老品牌名的好处；
2. 沿用老品牌名的风险；
3. 是否会稀释原品牌定位；
4. 建议：沿用 / 子品牌 / 新品牌；
5. 命名建议。
```
